# a-star-algorithm

Implementação de um algoritmo A* (A star)

Este algoritmo recebe um labirinto 2D, um **ponto de partida**, **um ponto de destino**, **onde ele pode "caminhar"**, e calcula a menor rota para chegar até o destino.
